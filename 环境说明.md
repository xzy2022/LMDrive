`/workspace/carla-0.9.10-py3.8-linux-x86_64.egg`被写入`/opt/conda/envs/lmdrive-py38/lib/python3.8/site-packages/carla_local.pth`中。

当前推理运行时不需要执行 `cd /workspace/LMDrive/vision_encoder && pip install -e .`。

- `vision_encoder/setup.py` 保留为旧训练链路的安装入口。
- 该安装会把仓库中的旧 fork 注册成包名 `timm`，覆盖当前环境里的官方 `timm`，不再作为推理环境的推荐路径。
- 当前推理链会直接使用项目内的本地 visual encoder factory。
- LiDAR 分支仍依赖 `torch_scatter`，需要宿主环境安装与系统 ABI 兼容的版本。

```bash
cd /workspace/LMDrive
```
